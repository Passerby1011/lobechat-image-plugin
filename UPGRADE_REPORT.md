# 🎨 LobeChat Image Plugins Hub 升级完成报告

本项目已根据 **2026-01-02** 提供的最新 API 文档完成全面升级。

## 🌟 升级亮点

### 1. 阿里百炼 (Tongyi & Qwen) - **全能影像中心**
- **同步/异步双引擎**：支持 `qwen-image-max` 等顶级同步模型，实现秒级高质量出图。
- **图像编辑与融合**：通过 `qwen-image-edit-plus` 支持多图风格迁移、换装及局部重绘。
- **高保真图像翻译**：新增 `qwen-mt-image`，支持多语言互译并精准保持原始排版。
- **最新版 Wan2.6**：全面支持最新的万相 2.6 系列模型。

### 2. 豆包 (Doubao) - **架构现代化与 Seedream 4.5**
- **Ark V3 迁移**：API 接口从旧版 CV 协议全面迁移至火山引擎 Ark V3 协议。
- **多图融合能力**：支持 Seedream 旗舰版的多图（最多14张）特征融合生成。
- **连贯组图生成**：支持漫画分镜、套图等 `sequential` 连贯生成模式。
- **SeedEdit 编辑**：引入专门的图像编辑模型 SeedEdit 3.0。

### 3. 腾讯混元 (Hunyuan) - **极速版 TextToImageLite**
- **秒级同步出图**：新增极速版同步接口，告别异步轮询等待。
- **3.0 专业版增强**：保留并优化了混元 3.0 异步任务处理逻辑，画质更精细。

### 4. 硅基流动 (SiliconFlow) - **顶级开源模型库**
- **FLUX 全系列**：涵盖 Schnell, Dev, Pro 三个版本。
- **中文优化 Kolors**：集成可图 (Kolors) 模型，提供最强的中文语义理解能力。
- **多样化模型**：支持 SD3 Medium、SDXL、Qwen-Image 等。

### 5. 智谱 AI (CogView) - **CogView-4 旗舰升级**
- **汉字生成优化**：升级至 CogView-4，显著提升画面中的汉字渲染准确度。
- **高清 (HD) 质量**：支持 HD 模式，提供更丰富的画面细节和光影表现。

---

## 🛠️ 核心代码改进

- **Handler 重构**：所有处理逻辑均已实现多模式（同步/异步/多模态）兼容。
- **统一响应格式**：标准化了 Markdown 返回格式，包含详细的模型、尺寸及参数信息。
- ** manifest 升级**：更新了所有插件的 `manifest.json` 到 V2/V3 版本，暴露了更多高级控制参数。

## 🚀 部署与使用建议

1. **更新环境变量**：
   - 建议在 LobeChat 客户端中更新设置。
   - 阿里插件现在统一使用 `ALIBABA_API_KEY`。
   - 豆包插件推荐使用新版 `ARK_API_KEY`。

2. **重新添加插件**：
   - 由于 API 参数和模型列表有重大变动，建议在 LobeChat 插件中心删除旧插件并根据新的 `manifest.json` 重新添加。
